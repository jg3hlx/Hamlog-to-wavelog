　Turbo HAMLOGのQSLカード印刷機能を使いwavelog https://github.com/wavelog/wavelog に取り込む為ためのファイルを作成します。

使用上の注意点

１．出力制限のため、一度に30000件以上の出力ができません。レコード番号で分割して出力してください。

２． ログに日本語が含まれている場合、出力されたファイルの文字コードを"UTF-8"に変更してください。
 
サンプル

　基本例　Hamlog to wavelog 1.qsl
 
 　QSLカードの受領情報例　Hamlog to wavelog 2.qsl

QSLカードの受領情報については各局の管理方法に合わせて変更してください。

wavelogへの取り込み

1.画面右側に表示されているコールサインをクリックする。

2.表示されたメニュー内の「ADIF インポート/エクスポート」をクリックする。

<img width="1161" height="555" alt="スクリーンショット 2026-05-03 220756" src="https://github.com/user-attachments/assets/127aa0b2-1e97-4ed8-8606-36621d5aaf8e" />

  
①取り込むステーション（ログファイル）を選択する。

②取り込みたいADIFファイルを選択する。

③アップロードをクリックして取り込みを開始する。
